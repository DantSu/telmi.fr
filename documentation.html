Telmi Sync : Installation

Pour télécharger et installer Telmi Sync, rendez-vous à cette adresse : https://telmi.fr/index.html#download

Windows :

  • Pour Windows, il est possible que vous deviez faire :
    • Clic droit sur le fichier d'installation de Telmi Sync téléchargé.
    • Cliquez sur Propriétés.
    • Cochez la case Débloquer situé en bas de la fenêtre des propriétés.
    • Validez la fenêtre des propriétés en cliquant en bas sur OK.
    • Maintenant vous pouvez double cliquer sur le fichier .exe de Telmi Sync pour lancer l'installation.
  • Le TTS (Text To Script) inclus dans Telmi Sync nécessite Microsoft Visual C++ Redistributable téléchargeable ICI.

Linux :

  • A partir de Ubuntu 24.04, l'argument --no-sandbox doit être ajouté au lancement de l'AppImage.

Telmi OS : Mettre à jour

La mise à jour de Telmi OS est automatique.

  • Ouvrez Telmi Sync
  • Connectez votre carte micro SD à l'ordinateur
  • Telmi Sync télécharge et installe la mise à jour automatiquement

Telmi OS : Mode nuit

Lorsque vous sélectionnez le dragon accompagné d'une lune et d'étoiles, vous entrez dans le Mode Nuit.

Vous vous retrouvez sur la liste des packs présent dans la Telmi avec un compteur en haut à gauche de l'écran (Un ovale orange avec une lune et 5 étoiles). Ce compteur vous indique le nombre d'histoire dans votre playlist.

Vous devez donc ensuite sélectionner une histoire comme dans le "Mode Histoires" classique sauf que lorsque une histoire (compatible avec le mode nuit) sera lancé, l'écran ne sera pas éteint. A La place il affichera à l'écran deux options :

  • Appuyer sur "A" pour ajouter une histoire à la playlist.
  • Appuyer sur "START" pour ajouter une histoire à la playlist et la lancer.

La playlist se lancera, toutes les histoires sélectionnées se liront à la suite les unes des autres. Une fois la playlist finit, la console s'éteindra.

ATTENTION : Les histoires compatibles avec le mode nuit sont les audio qui respectent les deux règles suivantes :

  • les audio en AUTOPLAY (qui passe automatiquement à l'audio suivant à la fin de celui-ci)
  • Les audio qui NE sont PAS passable avec le bouton A (Bouton OK non activé sur STUdio)

Cette fonctionnalité est désactivable via Telmi Sync.
