Telmi Sync : Installation

Pour télécharger et installer Telmi Sync, rendez-vous à cette adresse : https://telmi.fr/index.html#download

  • Pour Windows, il est possible que vous deviez faire :
    • Clic droit sur le fichier d'installation de Telmi Sync téléchargé.
    • Cliquez sur Propriétés.
    • Cochez la case Débloquer situé en bas de la fenêtre des propriétés.
    • Valider la fenêtre des propriétés en cliquant en bas sur OK.
    • Maintenant double cliquez sur le fichier d'installation de Telmi Sync.

Telmi OS : Installation (Partie 1 - Formatage FAT32)

Pour installer Telmi OS il faut commencer par formater une carte micro SD en FAT32.

Attention un formatage supprime toutes les données présentes sur la carte micro SD.

Windows : Formater en FAT32

Veuillez télécharger Rufus (https://rufus.ie/fr/) :

  • Branchez votre carte micro SD à l'ordinateur.
  • Lancer le logiciel Rufus.
  • Sélectionnez dans Périphérique la carte micro SD que vous souhaitez formater.
  • Sélectionnez dans Type de démarrage => Non démarrable
  • Sélectionnez dans Schéma de partition => MBR
  • Sélectionnez dans Système de destination => BIOS ou UEFI
  • Sélectionnez dans Système de fichiers => Large FAT32
  • Sélectionnez dans Taille d'unité d'allocation => 32 kilo-octets
  • Pour finir cliquez sur DEMARRER

Le formatage est terminé, maintenant rendez-vous un peu plus bas pour la suite !

Mac OS : Formater en FAT32

Sur MacOS vous pouvez formatter votre carte micro SD avec l'utilitaire de disque :

  • Branchez votre carte micro SD à votre Mac.
  • Ouvrez Application > Utilitaires > Utilitaire de disques
  • Dans le menu de gauche sélectionnez la carte micro SD que vous souhaitez formatter.
  • Cliquez sur le bouton Effacer en haut de la fenêtre.
  • Sélectionnez dans Format => MS-DOS (FAT32)
  • Cliquez sur Effacer pour valider

Le formatage est terminé, maintenant rendez-vous un peu plus bas pour la suite !

Linux : Formater en FAT32

Veuillez vous référer à ce tutoriel pour formater votre carte en FAT32 : https://www.golinuxcloud.com/steps-to-format-sd-card-in-linux/

Le formatage est terminé, maintenant rendez-vous un peu plus bas pour la suite !

Telmi OS : Installation (Partie 2)

Votre carte micro SD est maintenant formater en FAT32 et toujours connectée à votre ordinateur. Nous pouvons passer aux étapes suivantes :

  • Téléchargez le fichier pour la Miyoo Mini + disponible à cette adresse : https://telmi.fr/index.html#download
  • Dézippez le contenu du fichier téléchargé à la racine de votre carte micro SD.
  • Ejectez et retirez la carte micro SD de votre ordinateur.
  • Insérez la carte dans la Miyoo Mini + éteinte.
  • Allumez la Miyoo Mini +.
  • L'installation se lance automatiquement, veuillez patienter.

Telmi OS : Mettre à jour

La mise à jour de Telmi OS est automatique.

  • Ouvrez Telmi Sync
  • Connectez votre carte micro SD à l'ordinateur
  • Telmi Sync télécharge et installe la mise à jour automatiquement

Telmi OS : Mode nuit

Lorsque vous sélectionnez le dragon accompagné d'une lune et d'étoiles, vous entrez dans le Mode Nuit.

Vous vous retrouvez sur la liste des packs présent dans la Telmi avec un compteur en haut à gauche de l'écran (Un ovale orange avec une lune et 5 étoiles). Ce compteur vous indique le nombre d'histoire dans votre playlist.

Vous devez donc ensuite sélectionner une histoire comme dans le "Mode Histoires" classique sauf que lorsque une histoire (compatible avec le mode nuit) sera lancé, l'écran ne sera pas éteint. A La place il affichera à l'écran deux options :

  • Appuyer sur "A" pour ajouter une histoire à la playlist.
  • Appuyer sur "START" pour ajouter une histoire à la playlist et la lancer.

La playlist se lancera, toutes les histoires sélectionnées se liront à la suite les unes des autres. Une fois la playlist finit, la console s'éteindra.

ATTENTION : Les histoires compatibles avec le mode nuit sont les audio qui respectent les deux règles suivantes :

  • les audio en AUTOPLAY (qui passe automatiquement à l'audio suivant à la fin de celui-ci)
  • Les audio qui NE sont PAS passable avec le bouton A (Bouton OK non activé sur STUdio)

Cette fonctionnalité est désactivable via Telmi Sync.
